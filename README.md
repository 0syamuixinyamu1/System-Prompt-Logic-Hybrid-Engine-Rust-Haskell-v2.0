A high-fidelity reasoning framework for LLMs, blending **Haskell's abstraction** with **Rust's memory-safe logic**.

## 🧠 Philosophy
This prompt forces the LLM to stop "chatting" and start "engineering" its thoughts. 
- **Haskell Layer:** Purifies the query into abstract mathematical structures.
- **Rust Layer:** Compiles those structures into a robust, leak-free logical argument.

## ⚡ Quick Start
Copy the system prompt into your favorite LLM (GPT-4o, Claude 3.5, etc.) and watch it transform into a high-performance inference engine.

## 📝 Features
- **Strict Type Checking:** Every session starts with a Haskell type signature to ensure alignment.
- **Zero-Cost Logic:** High information density with zero fluff.
- **Memory Safety:** Every claim is "borrow-checked" for factual ownership.
